# Output contract

Use these shapes to keep communication consistent.

## Start confirmation

- objective
- current understanding / assumptions
- chosen execution mode
- first milestone

## Stage update

- completed
- now doing
- next risk or decision

## Blocker alert

Use this format:

- blocker
- impact
- options

Example options:

- continue deep investigation
- narrow scope and ship a smaller slice
- stop coding and deliver findings only

## Final delivery

- result summary
- assumptions
- changed files
- run commands
- verification results
- remaining gaps
- recommended next step
