# nextjs-nestjs-starter

Split full-stack starter with a Next.js client and a NestJS server.

## First slice included

- backend health route
- backend item list/create example
- frontend page that fetches items from NestJS

## Run

```bash
# server
cd server
npm install
npm run dev

# client
cd client
npm install
npm run dev
```
